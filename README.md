# AdsPilot - Shopee Ads Automation System

AdsPilot adalah ekosistem aplikasi untuk otomatisasi iklan Shopee, dirancang untuk skalabilitas tinggi dan efisiensi manajemen iklan.

## Arsitektur Sistem
Project ini terdiri dari beberapa komponen utama:
- `adm`: Portal Admin untuk manajemen user dan sistem.
- `aff`: Portal Affiliate untuk manajemen referral.
- `app`: Aplikasi utama/dashboard user.
- `landing-page-v2`: Landing page informasi dan marketing.
